# 麦肯锡PPT制作Skills库

## 概述

本Skills库萃取自麦肯锡咨询PPT的精髓，包含完整的PPT制作工作流方法论。通过系统化的工作流节点，帮助用户创建专业级的商业演示文稿。

**当前版本**: v2.2
**Skills数量**: 18个核心Skills
**特色功能**: 母婴跨境电商12大业务场景 + 8大多维分析图表 + 10大消费者分析图表 + 8大品牌分析图表 + 8大渠道分析图表

## 核心方法论

### 麦肯锡PPT四大原则
1. **结论先行** - 每页幻灯片必须有明确的核心观点
2. **数据驱动** - 用数据支撑观点，图表传达信息
3. **简洁专业** - 去除冗余，聚焦核心信息
4. **逻辑清晰** - 信息层级分明，论证逻辑严密

### 标准幻灯片结构
```
┌─────────────────────────────────────────┐
│ Slide Headline (标题 = 核心观点)          │
├─────────────────────────────────────────┤
│ 概述图表 (一句话说明图表核心内容)          │
├─────────────────────────────────────────┤
│ 通过图表可知 (3-5点核心结论)              │
│ • 结论1 - 数据支撑                        │
│ • 结论2 - 数据支撑                        │
│ • 结论3 - 数据支撑                        │
├─────────────────────────────────────────┤
│ 建议行动 / 值得关注                       │
├─────────────────────────────────────────┤
│ 关键词 (Key Terms + 数据来源)            │
├──────────────────────┬──────────────────┤
│                      │                  │
│    文字说明区域        │    图表区域      │
│    (2/3宽度)          │    (1/3宽度)     │
│                      │                  │
└──────────────────────┴──────────────────┘
```

## 新增Skills说明 (v2.0)

### 1. 复合图表 (mckinsey-combination-chart)
**用途**: 同时展示多个维度数据

**支持的组合类型**:
- 柱状图 + 折线图 (规模+增长率)
- 柱状图 + 饼图 (结构+占比)
- 面积图 + 折线图 (累积+趋势)

**母婴电商案例**:
- 品类销售与增长分析
- 渠道分布与转化率分析
- 价格带销售与毛利率分析

### 2. 分组柱状图 (mckinsey-grouped-chart)
**用途**: 多维度对比分析

**适用场景**:
- 时间对比 (2023 vs 2024)
- 群体对比 (不同城市层级)
- 属性对比 (不同特征)

**母婴电商案例**:
- 消费态度年度对比
- 品类购买转化率对比
- 渠道用户满意度对比
- 竞品价格带分布对比

### 3. 图表分类体系 (mckinsey-chart-classification)
**用途**: 系统性图表选择指南

**六大类别**:
1. **对比类**: 条形图、柱状图、分组图
2. **构成类**: 环形图、饼图、堆叠图
3. **趋势类**: 折线图、面积图、阶梯图
4. **关系类**: 散点图、气泡图、热力图
5. **分布类**: 直方图、箱线图、密度图
6. **复合类**: 柱状+折线、子弹图、仪表盘

### 4. 母婴电商图表应用 (mckinsey-maternal-ecommerce-charts)
**用途**: 母婴跨境电商12大业务场景图表模板

**业务场景覆盖**:
1. 品类销售分析
2. 渠道效果分析
3. 用户画像分析
4. 价格带分析
5. 促销效果分析
6. 用户留存分析
7. 库存周转分析
8. 竞品对比分析
9. 地域市场分析
10. 用户生命周期价值
11. 内容营销效果
12. 供应链效率

### 5. 多维分析图表 (mckinsey-multi-dimensional-charts) ✨NEW
**用途**: 复杂业务场景的多维对比分析

**8大图表类型**:
1. **增长驱动分解图** - 销额增长归因分析（堆叠柱状图/瀑布图）
2. **竞争转换分析图** - 消费者品牌流向分析（双向条形图）
3. **价格带分析图** - 高端化趋势分析（堆叠柱状图+条形图）
4. **消费者特征矩阵图** - 流失/新进买者画像（散点图+堆叠图）
5. **多属性对比图** - 品牌形象多维度对比（8图组合）
6. **时间趋势对比图** - 多指标趋势分析（折线+柱状图）
7. **瀑布图** - 财务分析/利润变化（瀑布图）
8. **矩阵气泡图** - 产品组合分析（气泡图）

**复合图表 vs 多子图表选择指南**:
- 2-3个维度，有关联 → 单一复合图
- 4+个维度，独立 → 多子图组合
- 单图表承载 ≤ 5个维度

### 6. 消费者分析图表 (mckinsey-consumer-analysis-charts) ✨NEW
**用途**: 消费者研究全场景图表方法论

**10大图表类型**:
1. **消费者构成变化图** - 用户健康度诊断（堆叠柱状图）
2. **人群画像对比图** - 品牌人群定位差异（堆叠条形图）
3. **重要性-表现分析图** - 改进优先级（散点图IPA）
4. **流失流向分析图** - 竞争威胁分析（双向条形图）
5. **用户分群雷达图** - 人群特征对比（雷达图）
6. **系列贡献人群分析图** - 产品人群定位（堆叠柱状图）
7. **购买驱动因素对比图** - 竞争优劣势（折线图）
8. **消费场景分析图** - 场景渗透机会（分组柱状图）
9. **渗透率-频次矩阵图** - 人群价值定位（散点图）
10. **用户生命周期价值图** - LTV分析（阶梯图）

### 7. 品牌健康度分析图表 (mckinsey-brand-health-charts) ✨NEW
**用途**: 品牌研究全场景图表方法论

**8大图表类型**:
1. **品牌健康度漏斗图** - 转化瓶颈分析（漏斗图）
2. **品牌定位感知图** - 差异化定位（散点图）
3. **品牌属性对比图** - 优劣势属性（8图组合）
4. **品牌竞争力矩阵图** - 市场地位评估（气泡图）
5. **NPS驱动因素分析图** - 忠诚度驱动（条形图+散点图）
6. **品牌认知来源分析图** - 渠道效率（堆叠柱状图）
7. **品牌情感连接图** - 品牌联想（条形图）
8. **品牌区域表现图** - 区域差异（地图热力图）

### 8. 渠道策略分析图表 (mckinsey-channel-strategy-charts) ✨NEW
**用途**: 渠道运营全场景图表方法论

**8大图表类型**:
1. **渠道效率矩阵图** - 多维度效率对比（6图组合）
2. **渠道组合变化图** - 结构变化趋势（堆叠柱状图）
3. **品牌渠道分布对比图** - 渠道策略差异（堆叠柱状图）
4. **促销结构分析图** - 促销类型效率（堆叠柱状图）
5. **渠道价格带分布图** - 高端化程度（堆叠柱状图）
6. **渠道品类偏好图** - 品类偏好差异（热力图）
7. **促销时间趋势图** - 促销波动分析（折线图）
8. **渠道获客效率图** - 成本质量平衡（气泡图）

## 设计标准速查

### 颜色标准
- **主色**: 麦肯锡蓝 (#051C2C)
- **辅助蓝**: #1E3A5F, #3A5F8A
- **强调色**: #FF6B35 (橙色)
- **文字色**: 黑色 (#000000) / 深灰 (#333333)
- **背景色**: 白色 (#FFFFFF)
- **中性色**: #CCCCCC (浅灰), #999999 (灰色)

### 字体标准
- **标题**: Arial Bold, 18-24pt
- **正文**: Arial Regular, 12-14pt
- **数据标签**: Arial, 10-12pt
- **注释**: Arial, 9-10pt

### 布局标准
- **左文字**: 2/3宽度
- **右图表**: 1/3宽度
- **边距**: 上下左右各0.5英寸
- **模块间距**: 0.25英寸

## 观点提炼公式

```
公式1: [主体]在[维度]上[状态/趋势]，建议[行动]
示例: 辅食品类增速领先达45%，建议加大资源投入

公式2: [A]与[B]相比[差异]，这意味着[含义]
示例: TikTok渠道ROI高于亚马逊14%，意味着内容营销效率更高

公式3: 基于[趋势/数据]，预计[结果]，建议[行动]
示例: 基于东南亚55%增速，预计2年内成为第二大市场，建议提前布局

公式4: [发现]表明[问题/机会]，应[行动]
示例: 缺货率5%表明库存管理存在问题，应增加安全库存
```
