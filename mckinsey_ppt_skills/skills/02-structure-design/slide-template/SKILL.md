# 麦肯锡幻灯片模板 Skill

## 概述

麦肯锡幻灯片遵循严格的结构规范，确保信息传递高效、逻辑清晰。每页幻灯片都采用统一的"标题-概述-结论-关键词-图表"结构。

## 标准幻灯片结构

### 完整结构图

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  Slide Headline (标题)                                   │
│  [核心观点，一句话陈述，左对齐，18-24pt]                   │
│                                                         │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  概述图表                                                │
│  [一句话说明图表展示的核心内容，左对齐，12-14pt]            │
│                                                         │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  通过图表可知                                             │
│  • [结论1 - 最重要的发现]                                 │
│  • [结论2 - 支撑性发现]                                   │
│  • [结论3 - 补充性发现]                                   │
│  • [结论4 - 可选]                                        │
│  [左对齐，12-14pt，项目符号]                              │
│                                                         │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  关键词/值得关注                                          │
│  [术语1] | [术语2] | [术语3] | 数据来源: [来源名称]        │
│ [左对齐，10-12pt]                                        │
│                                                         │
├────────────────────────────────┬────────────────────────┤
│                                │                        │
│    文字说明区域                  │    图表区域            │
│    (可选，补充说明)              │    (核心可视化)        │
│    宽度: 2/3                    │    宽度: 1/3           │
│                                │                        │
│                                │                        │
│                                │                        │
└────────────────────────────────┴────────────────────────┘
```

### 模块详解

#### 1. Slide Headline (标题)

**位置**: 页面顶部
**宽度**: 100%
**高度**: 约页面1/6

**内容规范**:
```
- 一句话陈述核心观点
- 直接回答"这页PPT要说明什么"
- 使用完整的陈述句
- 避免使用"介绍"、"分析"等中性词

好标题示例:
✓ "数字化获客策略驱动销售增长50%"
✓ "品牌重要性随消费者年龄显著上升"
✓ "视频会议在所有商业场景下更受偏好"

差标题示例:
✗ "销售数据分析"
✗ "消费者调研结果"
✗ "会议偏好分析"
```

#### 2. 概述图表

**位置**: 标题下方
**宽度**: 100%
**高度**: 约页面1/8

**内容规范**:
```
- 一句话说明图表内容
- 点明数据来源和时间范围
- 简要说明分析维度

示例:
"图表展示了2024年Q1-Q3各销售渠道的业绩表现(数据来源: 销售系统)"
"基于1000名受访者的调研结果，展示不同年龄段的消费偏好"
```

#### 3. 通过图表可知

**位置**: 概述下方
**宽度**: 100%
**高度**: 约页面1/3

**内容规范**:
```
- 3-5个核心结论
- 按重要性排序
- 每点一行，简洁有力
- 数据支撑观点

结构:
• [最重要的发现] - 直接支撑标题
• [第二重要发现] - 补充说明
• [第三重要发现] - 进一步支撑
• [可选] - 补充细节

示例:
• 70-80%的客户偏好远程沟通方式
• 视频会议在所有商业场景下均比电话更受欢迎
• 数字化转型已成为B2B销售的必然趋势
• 企业需加速适应这一变化以保持竞争力
```

#### 4. 关键词/值得关注

**位置**: 结论下方
**宽度**: 100%
**高度**: 约页面1/10

**内容规范**:
```
- 关键术语定义
- 重要概念解释
- 数据来源标注

格式:
[术语1] | [术语2] | [术语3] | 数据来源: [来源名称]

示例:
B2B销售互动 | 数字化转型 | 远程偏好 | 数据来源: McKinsey Survey 2024
```

#### 5. 图表区域

**位置**: 页面右侧
**宽度**: 约1/3
**高度**: 页面下半部分

**内容规范**:
```
- 单一核心图表
- 清晰的数据可视化
- 符合麦肯锡设计规范
- 必要的图例和标签

图表类型选择:
- 对比: 条形图/柱状图
- 占比: 环形图/堆叠图
- 趋势: 折线图
- 多指标: 组合图
```

## 布局比例规范

### 标准比例

```
页面总高度 = 100%

├─ 标题区域: 15-18%
├─ 概述区域: 10-12%
├─ 结论区域: 25-30%
├─ 关键词区域: 8-10%
└─ 图表区域: 35-40%
```

### 左右分割

```
页面总宽度 = 100%

├─ 左侧文字区: 60-65%
└─ 右侧图表区: 35-40%

特殊情况:
- 无图表: 文字区100%
- 大图表: 可调整为50:50
```

## 边距规范

```
页面边距:
- 上边距: 0.5英寸
- 下边距: 0.5英寸
- 左边距: 0.5英寸
- 右边距: 0.5英寸

模块间距:
- 标题与概述: 0.25英寸
- 概述与结论: 0.25英寸
- 结论与关键词: 0.25英寸
- 关键词与图表: 0.25英寸
```

```

```

## 完整模板示例

```markdown
---
标题: 数字化获客策略驱动销售持续增长
---

## 概述图表
图表展示了2024年Q1-Q3各销售渠道的业绩表现及增长趋势

## 通过图表可知
• Q3销售额达1500万，同比增长50%，超出预期20%
• 线上渠道贡献70%增长，验证数字化策略有效性
• 新客户获客成本下降30%，ROI显著提升
• 订阅客户占比从25%提升至40%，收入更稳定

## 关键词
数字化获客 | 线上渠道 | 订阅转化 | 数据来源: 销售系统 2024

## 图表
[条形图: 各季度销售额对比]
```

## 质量检查清单

```
□ 标题是否直接陈述核心观点？
□ 概述是否清晰说明图表内容？
□ 结论是否有3-5个要点？
□ 关键词是否定义了关键术语？
□ 数据来源是否标注？
□ 图表是否符合设计规范？
□ 布局比例是否合理？
□ 字体大小是否一致？
□ 边距是否统一？
□ 整体是否简洁专业？
```
