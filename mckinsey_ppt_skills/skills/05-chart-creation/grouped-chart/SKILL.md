# 麦肯锡分组柱状图制作 Skill

## 概述

分组柱状图(Grouped Bar Chart)是将多个柱子并排放置进行对比，是麦肯锡展示多维度对比分析的首选图表类型。

## 适用场景

### 最适合
- 时间对比(2017 vs 2019)
- 群体对比(不同城市层级)
- 属性对比(不同特征)
- A/B测试结果展示

### 不适合
- 组别超过4个
- 需要展示占比关系
- 数据差异过大

## 设计规范

### 1. 基本结构

```
          │  2017  2019
    80%  ┤  ▓▓▓▓  ████
    60%  ┤  ▓▓▓▓  ████  ▓▓▓▓  ████
    40%  ┤  ▓▓▓▓  ████  ▓▓▓▓  ████
    20%  ┤  ▓▓▓▓  ████  ▓▓▓▓  ████  ▓▓▓▓  ████
     0%  └─────────────────────────────────────
          一二线城市    三四线城市    农村

      ▓ 2017  █ 2019
```

### 2. 分组规则

**时间对比**:
```
规则: 旧数据在前，新数据在后
示例: 2017柱子 | 2019柱子

颜色配置:
• 旧数据: #CCCCCC (浅灰)
• 新数据: #051C2C (深蓝)

目的: 突出变化趋势
```

**群体对比**:
```
规则: 按逻辑顺序排列
示例: 一线城市 | 二线城市 | 三线城市

颜色配置:
• 主要群体: #051C2C
• 次要群体: #3A5F8A
• 对照群体: #CCCCCC

目的: 便于横向对比
```

### 3. 间距规范

```
组内间距: 柱子宽度的20-30%
组间间距: 柱子宽度的50-100%

示例:
│█ █│    │█ █│    │█ █│
 └─┘      └─┘      └─┘
组内      组间      组内
```

### 4. 数据标注

```
方式1: 柱顶标注(推荐)
┌─────────────────────────────┐
│    75%    82%               │
│    ████   ████              │
│    ████   ████   68%   72%  │
│    ████   ████   ████  ████ │
└─────────────────────────────┘

方式2: 仅标注关键变化
┌─────────────────────────────┐
│           +7pp ↑            │
│    ████   ████              │
│    ████   ████   ████  ████ │
│    ████   ████   ████  ████ │
└─────────────────────────────┘
```

## 母婴跨境电商应用案例

### 案例1: 消费态度年度对比

**业务场景**: 分析不同城市层级消费者态度变化

**数据结构**:
```
城市层级     2023年    2024年    变化
一二线城市   72%       78%       +6pp
三四线城市   58%       68%       +10pp
县域市场     45%       55%       +10pp
```

**幻灯片设计**:
```markdown
# 下沉市场消费升级态度变化显著，提升10个百分点

## 概述图表
图表展示了2023-2024年不同城市层级消费者对高端母婴产品的态度变化

## 通过图表可知
• 三四线城市和县域市场消费升级态度提升最显著(+10pp)
• 一二线城市态度提升相对稳定(+6pp)
• 下沉市场与一线城市态度差距在缩小(从27pp缩小到23pp)
• 建议加大下沉市场高端产品布局

## 值得关注
• 消费升级趋势: 下沉市场增速 > 一线城市
• 市场机会: 县域市场存在结构性升级机会

## 关键词
Consumer Attitude | City Tier | Premium Products
```

### 案例2: 品类购买转化率对比

**业务场景**: 对比不同品类在促销期与平销期的转化率

**数据结构**:
```
品类        平销期    促销期    提升
奶粉        2.5%      4.2%     +68%
纸尿裤      3.0%      5.5%     +83%
辅食        1.8%      4.8%     +167%
洗护用品    2.2%      6.0%     +173%
```

**幻灯片设计**:
```markdown
# 洗护用品促销期转化率提升最显著达173%

## 通过图表可知
• 洗护用品促销期转化率提升173%，促销敏感度最高
• 辅食品类提升167%，说明消费者对价格敏感
• 奶粉品类提升68%相对较低，属刚需品类
• 纸尿裤转化率绝对值最高，促销期达5.5%

## 业务建议
• 洗护/辅食品类适合高频促销策略
• 奶粉品类应注重品牌建设而非价格促销
• 纸尿裤可利用高转化特性做连带销售
```

### 案例4: 竞品价格带分布对比

**业务场景**: 对比本品牌与主要竞品的价格带分布

**数据结构**:
```
价格带      本品牌    竞品A     竞品B
$0-20       20%       35%       15%
$20-50      45%       40%       35%
$50-100     25%       20%       35%
$100+       10%       5%        15%
```

**幻灯片设计**:
```markdown
# 本品牌中端($20-50)占比最高45%，竞品B高端($100+)布局领先

## 通过图表可知
• 本品牌$20-50价格带占比45%，是核心竞争区间
• 竞品A低价策略明显，$0-20占比35%
• 竞品B高端布局领先，$100+占比15%
• 本品牌$100+占比仅10%，高端市场存在机会

## 战略建议
• 巩固$20-50中端市场优势
• 考虑开发$50-100价格带产品
• 针对竞品A差异化定价
```

## 技术实现

### Python (Matplotlib)

```python
import matplotlib.pyplot as plt
import numpy as np

# 数据
categories = ['一二线城市', '三四线城市', '县域市场']
data_2023 = [72, 58, 45]
data_2024 = [78, 68, 55]

x = np.arange(len(categories))
width = 0.35

fig, ax = plt.subplots(figsize=(10, 6))

# 绘制分组柱状图
bars1 = ax.bar(x - width/2, data_2023, width, label='2023年', color='#CCCCCC')
bars2 = ax.bar(x + width/2, data_2024, width, label='2024年', color='#051C2C')

# 设置标签
ax.set_ylabel('态度占比(%)')
ax.set_xticks(x)
ax.set_xticklabels(categories)
ax.legend()

# 添加数值标注
def add_labels(bars):
    for bar in bars:
        height = bar.get_height()
        ax.annotate(f'{height}%',
                    xy=(bar.get_x() + bar.get_width() / 2, height),
                    xytext=(0, 3),
                    textcoords="offset points",
                    ha='center', va='bottom')

add_labels(bars1)
add_labels(bars2)

# 添加变化标注
for i, (old, new) in enumerate(zip(data_2023, data_2024)):
    change = new - old
    ax.annotate(f'+{change}pp',
                xy=(x[i] + width/2, new + 5),
                ha='center',
                color='#FF6B35',
                fontweight='bold')

plt.title('消费升级态度年度对比')
plt.tight_layout()
plt.show()
```
