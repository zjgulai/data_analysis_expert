# 麦肯锡图表分类体系 Skill

## 概述

麦肯锡图表分类体系按照分析目的和数据关系，将图表分为6大类别，帮助快速选择最合适的可视化方式。

## 六大图表类别

### 1. 对比类图表 (Comparison)

**核心目的**: 展示差异、排名、优劣

```
┌─────────────────────────────────────────────────────────┐
│ 对比类图表                                              │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  条形图 (Bar Chart)          柱状图 (Column Chart)      │
│  ┌──────────────┐            ┌────────────────────┐     │
│  │ ████████ 75% │            │ 75% ████           │     │
│  │ ██████ 60%   │            │ 60% ████  ████     │     │
│  │ ████ 45%     │            │ 45% ████  ████  ██ │     │
│  └──────────────┘            └────────────────────┘     │
│  横向对比/排名               纵向对比/时间趋势           │
│                                                         │
│  分组柱状图 (Grouped)        分组条形图 (Grouped Bar)   │
│  ┌────────────────────┐      ┌──────────────┐           │
│  │ ██ ██  ██ ██  ██   │      │ ██████ ██ 75%│           │
│  │ ██ ██  ██ ██  ██   │      │ ████   ██ 60%│           │
│  └────────────────────┘      └──────────────┘           │
│  多维度对比                   多维度横向对比              │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**适用场景**:
- 销售排名对比
- 渠道效果对比
- 竞品对比分析
- A/B测试结果

│  堆叠柱状图 (Stacked)       100%堆叠图 (100% Stacked)   │
│  ┌────────────────────┐      ┌────────────────────┐     │
│  │ ▓▓▓▓▓▓             │      │ ▓▓▓▓▓▓░░░░████    │     │

**适用场景**:

## 母婴跨境电商图表映射

### 按业务场景分类

| 业务场景 | 推荐图表 | 示例应用 |
|---------|---------|---------|
| 销售排名 | 条形图 | 品类销售TOP10 |
| 同比环比 | 柱状图+折线 | 销售额与增长率 |
| 市场份额 | 环形图 | 各渠道销售占比 |
| 用户画像 | 堆叠图 | 年龄段消费偏好 |
| 趋势分析 | 折线图 | 月度销售走势 |
| 渠道对比 | 分组柱状图 | 各渠道转化率对比 |
| 价格分布 | 直方图 | 订单金额分布 |
| 品类关联 | 热力图 | 跨品类购买关联 |
| KPI达成 | 子弹图 | 目标vs实际 |
| 用户分群 | 散点图 | RFM分群可视化 |

### 按数据类型分类

| 数据类型 | 推荐图表 | 特点 |
|---------|---------|------|
| 单维度排名 | 条形图 | 横向易读 |
| 时间序列 | 折线图/柱状图 | 展示变化 |
| 占比结构 | 环形图/堆叠图 | 部分-整体 |
| 多维度对比 | 分组柱状图 | 并排对比 |
| 相关性 | 散点图 | 展示关系 |
| 分布特征 | 直方图/箱线图 | 统计特性 |
| 规模+效率 | 复合图表 | 双轴展示 |

## 质量检查清单

```
□ 图表类型是否匹配分析目的？
□ 数据维度是否清晰展示？
□ 是否避免了过度设计？
□ 颜色是否易于区分？
□ 标题是否表达核心观点？
□ 数据来源是否标注？
```
