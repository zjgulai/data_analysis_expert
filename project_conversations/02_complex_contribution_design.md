## 二、现有能力分析

### 2.1 现有归因分析Skills

#### contribution-calculation (贡献度计算)
- **支持**: 加法型、乘法型、复合型指标
- **方法**: 链式替代法、Shapley值
- **局限**: 无法处理复杂混合运算

#### margin-attribution (毛利率归因)
- **支持**: 毛利率变动的多维归因
- **方法**: 结构变动vs自身变动分解
- **局限**: 专注于毛利率场景

#### binning-contribution (分箱贡献分析)
- **支持**: 按区间划分的贡献分析
- **方法**: 静态贡献+动态变动贡献
- **局限**: 需要先进行分箱处理

### 2.2 能力缺口

```
现有能力                          缺失能力
```

## 三、解决方案设计

### 3.1 数学基础

#### 偏微分（静态贡献度）

对于多变量函数 Y = f(X₁, X₂, ..., Xₙ)：

```
∂Y/∂Xᵢ 表示 Xᵢ 对 Y 的边际影响

业务含义: Xᵢ 变动1单位，Y 变动 (∂Y/∂Xᵢ) 单位
```

#### 弹性系数

```
Eᵢ = (∂Y/∂Xᵢ) × (Xᵢ/Y)

业务含义: Xᵢ 变动1%，Y 变动 Eᵢ%
```

#### 全微分（波动贡献度）

```
dY = (∂Y/∂X₁)dX₁ + (∂Y/∂X₂)dX₂ + ... + (∂Y/∂Xₙ)dXₙ

离散形式:
ΔY ≈ Σ(∂Y/∂Xᵢ) × ΔXᵢ

因素i的波动贡献度 = (∂Y/∂Xᵢ) × ΔXᵢ
因素i的贡献占比 = 波动贡献度 / ΔY × 100%
```

### 3.2 常见公式的偏导数

```python
# 公式1: 乘法链式 Y = A × B × C
∂Y/∂A = B × C
∂Y/∂B = A × C
∂Y/∂C = A × B
# 弹性: E_A = E_B = E_C = 1

# 公式2: 减法链式 Y = 1 - A - B - C
∂Y/∂A = -1
∂Y/∂B = -1
∂Y/∂C = -1

# 公式3: ROI型 Y = (A - B) / B
∂Y/∂A = 1/B
∂Y/∂B = -A/B²

# 公式4: 混合型 Y = (A × B + C) / D
∂Y/∂A = B/D
∂Y/∂B = A/D
∂Y/∂C = 1/D
∂Y/∂D = -(A×B+C)/D²

# 公式5: 毛利型 Y = A × (1 - B) - C
∂Y/∂A = 1 - B
∂Y/∂B = -A
∂Y/∂C = -1
```

### 3.3 交互效应处理

当多因素同时变动时，一阶全微分会有残差：

```
残差 = ΔY实际 - Σ一阶贡献

如果残差显著（>|10%|），说明存在交互效应
```

**交互效应来源**:
- 二阶效应: ∂²Y/(∂Xᵢ∂Xⱼ) × ΔXᵢ × ΔXⱼ
- 高阶效应: 更复杂的交叉项

**处理方式**:
1. 按比例分配给各因素
2. 单独报告交互效应
3. 使用二阶泰勒展开

## 五、案例验证

### 案例1: ROI变动归因

**公式**: ROI = (销售额 - 成本) / 成本

**数据**:
| 指标 | 上期 | 本期 | 变动 |
|------|------|------|------|
| 销售额 | 250万 | 280万 | +30万 |
| 成本 | 100万 | 140万 | +40万 |
| ROI | 2.5 | 1.0 | -1.5 |

**计算**:
```
∂ROI/∂销售额 = 1/成本 = 1/100 = 0.01
∂ROI/∂成本 = -销售额/成本² = -250/10000 = -0.025

销售额贡献 = 0.01 × 30 = +0.3
成本贡献 = -0.025 × 40 = -1.0
一阶合计 = -0.7

交互效应 = -1.5 - (-0.7) = -0.8 (53%)
```

**结论**: 成本上升贡献67%，交互效应53%，销售额增长抵消20%

### 案例2: 毛利额四因素归因

**公式**: 毛利额 = 访客数 × 转化率 × 客单价 × 毛利率

**数据**:
| 因素 | 上期 | 本期 | 变动率 |
|------|------|------|--------|
| 访客数 | 10万 | 12万 | +20% |
| 转化率 | 5% | 4.5% | -10% |
| 客单价 | 100 | 110 | +10% |
| 毛利率 | 30% | 28% | -7% |
| 毛利额 | 15万 | 16.6万 | +11% |

**计算**:
```
∂Y/∂访客数 = 转化率 × 客单价 × 毛利率 = 0.05 × 100 × 0.3 = 1.5
∂Y/∂转化率 = 访客数 × 客单价 × 毛利率 = 10万 × 100 × 0.3 = 300万
∂Y/∂客单价 = 访客数 × 转化率 × 毛利率 = 10万 × 0.05 × 0.3 = 1500
∂Y/∂毛利率 = 访客数 × 转化率 × 客单价 = 10万 × 0.05 × 100 = 50万

访客数贡献 = 1.5 × 2万 = +3万 (187.5%)
转化率贡献 = 300万 × (-0.5%) = -1.5万 (-93.8%)
客单价贡献 = 1500 × 10 = +1.5万 (93.8%)
毛利率贡献 = 50万 × (-2%) = -1万 (-62.5%)

一阶合计 = +2万
交互效应 = 1.6万 - 2万 = -0.4万 (-25%)
```

## 六、技术要点总结

### 6.1 核心公式

| 概念 | 公式 | 用途 |
|------|------|------|
| 偏导数 | ∂Y/∂Xᵢ | 静态敏感度 |
| 弹性系数 | (∂Y/∂Xᵢ)×(Xᵢ/Y) | 跨因素比较 |
| 波动贡献 | (∂Y/∂Xᵢ)×ΔXᵢ | 绝对贡献 |
| 贡献占比 | 波动贡献/ΔY | 相对贡献 |
| 交互效应 | ΔY - Σ一阶贡献 | 协同效应 |

### 6.2 实现要点

1. **在基期值处计算偏导数**: 保证一致性
2. **验证计算**: Σ贡献 + 交互效应 = 实际变动
3. **弹性系数标准化**: 便于跨因素比较
4. **交互效应阈值**: >10%需单独报告

### 6.3 适用场景

| 场景 | 适用性 | 说明 |
|------|--------|------|
| 简单乘法 | ✓ | 与链式替代法结果相近 |
| 复杂混合运算 | ✓✓ | 本方法优势 |
| 高精度需求 | ✓✓ | 精确计算交互效应 |
| 快速估算 | ○ | 可用简化方法 |
